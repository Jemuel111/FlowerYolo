About FloraLens

ITST 303 — Group Performance Task #3 · Object Detection System using YOLOv8 + Flask

What This System Does

FloraLens is an AI-powered flower recognition system that uses Ultralytics YOLOv8 to classify flower images into one of five categories: Daisy, Dandelion, Rose, Sunflower, and Tulip.

Users upload a photo through the web interface, and the system runs YOLO inference to return the top predictions with confidence scores, bounding boxes, and descriptive metadata about each flower.

Technology Stack

🤖

Ultralytics YOLOv8

State-of-the-art object detection & classification model trained on the Kaggle Flowers Recognition dataset.

🐍

Flask (Python)

Lightweight Python web framework handling image uploads, routing, and rendering Jinja2 templates.

📊

Kaggle Dataset

4,242 flower images across 5 classes — Daisy, Dandelion, Rose, Sunflower, and Tulip.

🎨

Jinja2 Templates

Server-side templating for dynamic result rendering — upload form, result images, and prediction scores.

How Inference Works

1. User uploads a flower image via the web form.

2. Flask saves the image to static/uploads/ with a unique timestamp filename.

3. YOLOv8 runs inference — returning class probabilities (classification) or bounding boxes (detection).

4. The annotated result image is saved to static/results/ and rendered back to the user with top-5 confidence scores.
